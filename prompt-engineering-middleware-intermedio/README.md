# Assignment: Prompt Engineering Avanzado - Middleware

Este repositorio contiene las plantillas de prompts, esquemas y estructura solicitada para el assignment de Prompt Engineering Avanzado en la vertical de Middleware & Backend. Todos los archivos están listos para que pegues los outputs generados por el LLM (Gemini u otro) y organices la evidencia según las instrucciones.

## Estructura
- `prompts/`: Prompts listos para usar en cada caso.
- `outputs/`: Carpetas donde deberás colocar los resultados generados por el LLM (se incluyen subcarpetas vacías con `.gitkeep`).
- `openapi-specs/`: Ubicación para las 5 especificaciones OpenAPI proporcionadas por el curso. Se incluyen archivos placeholder que debes reemplazar con las especificaciones reales.
- `screenshots/`: Espacio para capturas de pantalla de las ejecuciones en la Cell CLI.

## Uso
1. Abre cada archivo en `prompts/` y ejecuta el prompt correspondiente en tu LLM preferido.
2. Pega los outputs en las secciones marcadas dentro de cada archivo.
3. Coloca las evidencias (outputs, screenshots, reportes) en las carpetas de `outputs/` y `screenshots/` siguiendo la estructura requerida para el ZIP final.

## Nota
No se incluyen resultados generados por LLM. Todos los espacios de output se mantienen como placeholders para que completes el assignment con tus propias ejecuciones.
